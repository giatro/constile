Elenco degli articoli pubblicati

Come integrare (X)HTML, CSS e JavaScript per aggiungere un po' di sale al proprio sito web. Di seguito è riportato l'elenco degli articoli pubblicati in ordine cronologico.

  • Skin alternative in JS/CSS: Dare la possibilità agli utenti di scegliere l'aspetto del sito è certamente un bel servizio, capace anche migliorare l'accessibilità per gli utenti con problemi alla vista, come gli ipovedenti. Ciò è possibile grazie ai CSS e al DOM.
  • Menu ad albero in JS/CSS: Realizzeremo un menu ad albero sfruttando i CSS e il DOM attraverso il JavaScript. La semplicità del CSS e degli script contrastano con l'efficacia del risultato.
  • Ridimensionare i caratteri con un click: In questo articolo vedremo come migliorare l'accessibilità di un sito consentendo agli utenti di ridimensionare i caratteri con un click, facendo uso dei CSS e del DOM tramite JavaScript.
